# Nature Skills 详细功能说明

`nature-skills` 提供了一套完整的科研全流程 AI 辅助工具集。所有功能模块都存放在 `skills/` 目录下，并以 `nature-*` 命名。

## 核心设计原则

1. **唯一权威信源**：所有规则均基于《自然》系列期刊已发表的内容或官方指南，而非模糊的个人偏好。
2. **显式规则重于隐式**：每条规则都附带理由，而不仅是直接命令。
3. **上下文感知**：理解学术写作与图表的局部语境，针对论文的不同部分采用不同的逻辑。
4. **结果导向（Output-first）**：直接生成最终可用的成果（如文本、SVG图表、PPTX），拒绝产生不必要的中间规划文件。
5. **高可扩展性**：每个技能彼此独立，新增技能不会影响现有模块。

## 功能模块详述

以下是当前包含的 10 个核心能力模块：

### 1. 论文插图生成 (nature-figure)
- **功能**：生成符合《自然》期刊视觉标准的 Matplotlib 多面板图表（SVG格式）。
- **特点**：确保正确的排版体系、语义色彩搭配以及非冗余的面板信息架构（如强制使用 Arial/DejaVu Sans，文本保留为 svg `<text>` 节点）。内置图表家族包括柱状图、折线趋势、热力图、雷达图、3D球体等。

### 2. 学术语言润色 (nature-polishing)
- **功能**：将学术草稿（或中文草稿）转化为标准的《自然》级别英文散文。
- **特点**：强制执行低于 30 词的短句结构、准确的学术词汇、英式英语（British English），并针对“摘要”、“讨论”等不同章节动态调整时态与对冲语言（Hedging），同时防范夸大其词。

### 3. 学术章节写作 (nature-writing)
- **功能**：基于作者提供的论点、结果、笔记或中文草稿，撰写或重构整段学术章节。
- **特点**：遵循“证据优先”原则，杜绝伪造数据。严格把控从“领域背景”到“当前瓶颈”，再到“本研究填补空白”的递进逻辑。可处理摘要、引言、方法、结果及讨论等。

### 4. 模拟审稿评估 (nature-reviewer)
- **功能**：从独立审稿人的视角进行预投递稿件的审阅评估。
- **特点**：能够输出 3 份独立的审稿人意见报告，以及一份交叉评估总结。评估维度覆盖创新性、重要性、技术严谨度、展示清晰度及编辑部拒稿风险。

### 5. 严格格式引文生成 (nature-citation)
- **功能**：将文稿声明转化为符合 Nature 或 CNS (Cell, Nature, Science) 级别的引用候选，并导出为文献管理软件可读的 `ENW`, `RIS` 或 `Zotero RDF` 文件。
- **特点**：基于 Crossref 等真实元数据，拒绝幻觉与伪造。支持将中文学术主张转换为英文学术概念并进行精确检索匹配。

### 6. 数据可用性与 FAIR 审查 (nature-data)
- **功能**：为稿件准备“数据可用性声明”（Data Availability Statement）、存储库计划、数据集引用及 FAIR（可发现、可访问、可互操作、可重用）元数据审查。
- **特点**：双语感知，能够将中文的意图（如“合理请求”、“原始数据”）翻译为符合施普林格·自然 (Springer Nature) 政策的严谨英文声明。

### 7. 论文审稿回复信 (nature-response)
- **功能**：起草并审计逐条回复审稿人的回复信（Response Letter / Rebuttal）。
- **特点**：为每条意见分配唯一 ID，将其分类并映射到具体的修改动作（如 `ACCEPT_TEXT`, `SOFTEN_CLAIM` 等），并强制要求在文中附带修改位置（行号或章节），保证回复信有理有据、态度合作。

### 8. 双语 Markdown 沉浸式阅读 (nature-reader)
- **功能**：全篇论文的双语 Markdown 阅读器转换。
- **特点**：支持原文对照、图文对应，通过在文本中加入源码锚点，提供极佳的文献阅读体验。

### 9. 论文转学术 PPT (nature-paper2ppt)
- **功能**：将论文、预印本、摘要或阅读笔记提炼为全中文的 `.pptx` 汇报演示文稿。
- **特点**：针对组会、文献汇报（Journal Club）设计。以科学逻辑为主轴抽取核心图表与证据，拒绝暴力压缩图片。产出包含 PPT 页面及详细的演讲者备注。

### 10. 多源学术检索与管理 (nature-academic-search)
- **功能**：通过本地 MCP 服务提供 PubMed, CrossRef 和 arXiv 的并行学术检索。
- **特点**：支持按 DOI, PMID 获取元数据，具备去重功能，并支持输出 `.nbib`, `.ris`, `.bib` 格式。对医学检索支持 MeSH 词表扩展。

---

通过灵活运用以上模块，用户可以全面提升从文献调研、构思、绘图到初稿写作、润色与审稿回复的科研工作质量，直接对标世界一流的学术标准。
